# Public Washroom App

## Setup
Get a Google Maps API key and put it in a .env file in the client folder for the front end to load
```
VITE_GOOGLE_MAPS_API_KEY=<Your API Key Here>
```

In terminal 1
```
cd server
npm install
npm start
```

In terminal 2
```
cd client
npm install
npm run dev
```

Point your browser at localhost:5173
